# JS Applications
## Course description
In the "JS Applications" course, students will learn what HTTP requests are and how to use them. They will become familiar with REST services, what a BaaS (Backend as a Service) is and how to work with it, what asynchronous code means (Promises, using async/await), and what templating and routing are. During the course, they will build a Single Page Application using the techniques learned in previous lectures. They will understand the architecture of an application and how to properly structure their applications. Toward the end of the course, they will explore various design patterns and their practical applications, create their own web components using the Web Components standard, and set up a Webpack environment from scratch. In addition, 30% of the exercise tasks will be solved with the help of AI, with the aim of encouraging the use of modern technologies for process automation while also developing skills for effectively applying AI tools in real-world conditions.
## Course content
* HTTP and REST Services
* Asynchronous Programming
* Remote Data and Authentications
* Single Page Applications
* Architecture and Testing
* Client Side Rendering
* Routing
* Modular Applications
* Exam Preparation
* Workshop
* Workshop: Web Components

## Acquired skills
* Working with HTTP and how to make requests
* Creating simple Single Page Applications
* Working with templating and routing
* Working with non-relational databases
* Working with REST APIs and asynchronous programming
* Testing the application
